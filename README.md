# LeadPilot SaaS

Votre SaaS de génération de leads avec Next.js, Supabase et Stripe.

## Installation

1. Clonez le repository
2. Installez les dépendances : `npm install`
3. Configurez les variables d'environnement
4. Lancez le serveur : `npm run dev`

## Technologies

- Next.js 14
- Supabase
- Stripe
- Tailwind CSS
- TypeScript
